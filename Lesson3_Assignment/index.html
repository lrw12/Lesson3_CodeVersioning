Highlight #1 Local Repository

A local respository is initially a copy of your data and installations which is stockpile of the remote downloads. It contains your information that you may not be ready to publish or release.

Highlight #2 Version Control

Version control is the system that keeps track of the changes you make to your files and can later go back and recall what has been done. There are also different types of Control Systems; Local Version, Centralized Version and Distributed Version. All have the same purpose, but work a little differently than each other.

Highlight #3 GIT Push

GIT Push is a command to push your data up that is not in the remote repository. This doesn't correspond with the command "commit". Although, you may have many commit commands beofre you push or fetch your files. GIT push is the command to share your files.
